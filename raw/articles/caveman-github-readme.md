# Caveman: why use many token when few token do trick
Claude Code skill/plugin that cuts ~75% of output tokens.
Levels: Lite, Full, Ultra, Wenyan.
Sub-skills: caveman-commit (terse commits), caveman-review (one-line reviews), caveman-compress (compresses CLAUDE.md memory files).
Installation: claude plugin marketplace add JuliusBrussee/caveman && claude plugin install caveman@caveman
