# API NFSe Nacional: o que é e como integrar?
A API NFSe Nacional marca uma revolução na emissão de Notas Fiscais de Serviço Eletrônicas (NFSe) no Brasil. Antes, empresas e desenvolvedores enfrentavam grandes desafios devido à falta de padronização entre os municípios.
Com o lançamento do Padrão Nacional de NFSe, a Receita Federal e o Serpro introduziram uma solução unificada.
Novidades:
- Login integrado com GOV.BR para MEIs (Selo Prata ou Ouro).
- Emissão simplificada para MEIs (apenas 3 informações).
- DANFSE em PDF no mobile.
- Obrigatoriedade para MEI desde 1º de setembro de 2023.
A API conecta sistemas de gestão ao Padrão Nacional. Recursos: Emissão, Consulta, Cancelamento, Eventos.
Integração via REST HTTPS.
Etapas: Credenciamento, Escolha da API (ex: TecnoSpeed), Configuração de ambiente, Testes e Implantação.
