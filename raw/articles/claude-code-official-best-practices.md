# Best Practices for Claude Code - Official Docs

Claude Code is an agentic coding environment.

## Key Strategies
- Give Claude a way to verify its work (tests, screenshots).
- Explore first, then plan, then code.
- Provide specific context in prompts (@ files, URLs).
- Manage context aggressively (performance degrades as it fills).
- Use /init to generate CLAUDE.md.
- Use auto mode for permissions.
